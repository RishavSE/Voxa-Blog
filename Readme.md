<h2 align="center">📌 Overview</h2>

Voxalog is a feature-rich blogging platform built using the MERN stack, designed to provide a seamless content creation and interaction experience.

The platform integrates secure authentication, rich text editing, and real-time engagement into a unified ecosystem.

It focuses on delivering a smooth user experience, scalability, and interactive communication between users.

<h2 align="center">🧠 Vision</h2> <p align="center"> <b>Content creation should be simple, expressive, and interactive.</b> </p>

Voxalog aims to:

• Enable effortless blog creation
• Encourage social engagement
• Provide a secure user environment
• Deliver a clean and responsive experience
• Build a scalable content-sharing ecosystem

<h2 align="center">🔐 Authentication & Access Control</h2>

Voxalog ensures strong security and access control using modern authentication techniques.

Features

• JWT-based authentication
• Password hashing using bcrypt
• Secure login and registration
• Forgot/reset password via modal
• Protected routes
• Role-based access (User/Admin – upcoming)

<h2 align="center">📝 Blog Management System</h2>

Voxalog provides a powerful blog creation and management system.

Capabilities

• Create and publish blogs
• Edit and update blogs
• Delete blogs anytime
• View trending and latest blogs

Rich Text Editor Features

• Bold, italic, underline
• Text color and highlight
• Text alignment (left, center, right)
• Image and video uploads

<h2 align="center">🌐 Social Interaction Engine</h2>

The platform promotes user engagement through interactive features.

Features

• View all public blogs
